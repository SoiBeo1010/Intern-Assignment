# Frontend

This is the React + Vite frontend for G-Scores.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

The app expects the backend API to be available at `http://localhost:4000`.
Vite proxies `/api` requests to that backend during development.

## Features

- Search a student by registration number
- View score distribution by subject and score band
- See the top 10 Group A students# Frontend (suggested)

This folder is a placeholder for the frontend app. Recommended quick start with React + Vite or Create React App.

Example to scaffold with Vite + React + TypeScript:

```
npm create vite@latest frontend -- --template react-ts
cd frontend
npm install
npm run dev
```

Place UI code under `frontend/src` and static `public` assets under `frontend/public`.
